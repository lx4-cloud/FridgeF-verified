# FridgeF Verified Page

## Quick Start

```bash
# 1) Clone and install
npm i

# 2) Run locally
npm run dev

# 3) Deploy to Vercel
```

## URL Behavior
- Page lives at `/verified`.
- Supports `?next=/login` (auto-redirect in ~1.8s).

## Customization
- Replace image URLs in `app/verified/page.tsx` with your assets.
- Update brand colors in `tailwind.config.ts`.
